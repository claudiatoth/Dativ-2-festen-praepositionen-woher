Verbe verificate pe PONS (de.pons.com/verbtabellen):
- **fahren** 🚨 TARE + SEIN (fuhr · ist gefahren) — mit/nach + Dativ
- **wohnen** REGULAT (wohnte · hat gewohnt) — bei/seit + Dativ
- **kommen** 🚨 TARE + SEIN (kam · ist gekommen) — aus/von + Dativ
- **gehen** 🚨 TARE + SEIN (ging · ist gegangen) — zu + Dativ
- **arbeiten** REGULAT -t Stamm (arbeitete · hat gearbeitet) — bei/mit + Dativ
- **bekommen** 🚨 TARE + **HABEN** (bekam · hat bekommen) — false friend „become" englezesc! = a PRIMI + von + Dativ
